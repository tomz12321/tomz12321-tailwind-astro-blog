# Day #37

## 前言

長程航班不是寫作的敵人，而是把輸出切成可攜帶模組的壓力測試環境。

## 波哥大初印象

09/13 抵達波哥大（IST → BOG 15:05 落地）之後，我讓自己先穩住節奏，再把旅行感受與工作節律連上線。這一篇用「目標／阻礙／行動／結果」梳理落地第一晚與舊城區一日遊的重點。

## 抵達與入住

- 抵達時間：9/13（六）當地 15:05
- 住宿：IHG 集團飯店（近主要動線、步行可達交通樞紐）
- 目的：先建立「安全・補水・作息校正」三項基線，避免剛到新城市就把體力花在非必要的奔波上。

---

## 目標 (Goal)

1. 在 24 小時內把「旅行狀態」切換成「可穩定輸出的生活狀態」。
2. 以舊城區為主場，建立一條可重複的步行弧線：廣場 → 市集 → 巷弄咖啡 → 高地視角。
3. 以 AI 協作做「敘事提要」而非代寫，確保素材與語氣仍是自己的。
4. 為後續長篇整理鋪路：記錄 10+ 個具體觀察點與 3 個可延展主題。

---

## 阻礙 (Obstacles)

| 類型       | 描述                           | 影響                         |
| ---------- | ------------------------------ | ---------------------------- |
| 海拔與氣候 | 波哥大高海拔＋午後雷陣雨機率   | 走路容易喘、行程需彈性調整   |
| 語言切換   | 西語主導、英文比例較低         | 資訊取得慢、溝通需手勢輔助   |
| 安全邊界   | 舊城區人潮密集、口袋防護需注意 | 攝影與筆記同時進行的風險上升 |
| 生理節奏   | 長程後的晚間進食與睡眠時間漂移 | 早晨專注度不穩               |

---

## 行動 (Actions)

### 1) 落地當晚（三步）

- 補水與碳水：先喝 500ml 水＋輕食，避免暴飲暴食。
- 簡單校正：熱水澡 + 15 分鐘伸展，讓身體從「飛行型緊繃」轉回「地面節奏」。
- 輕量筆記：列 6 個 Day 37 想回答的問題，睡前不開長文編輯工具。

### 2) 舊城區一日弧線

```
起點：主要廣場（聲音密度 + 節奏感）
→ 市集軸線（氣味與色彩）
→ 巷弄咖啡（20 分鐘定點：把條列轉為三行敘事）
→ 博物館或高地視角（拉遠城市紋理）
→ 黃昏返回（光線轉換與安全評估）
```

### 3) 感官採集與最小敘事

| 感官 | 提示               | 記錄（節選）                                     |
| ---- | ------------------ | ------------------------------------------------ |
| 聲音 | 節拍 / 呼喊 / 靜點 | 廣場鼓聲與街頭叫賣互為背景，巷內有可貴的「靜點」 |
| 氣味 | 固定 vs 突發       | 玉米餅與咖啡香穩定，雨前土腥味突發               |
| 地面 | 材質 → 步頻        | 石板濕滑，步伐自然放慢，拍照節點變少             |
| 光線 | 角度 / 反射        | 午後雲層厚、建物色彩偏冷；黃昏反射讓牆面更立體   |
| 互動 | 商店 / 問路        | 簡單西語 + 手勢可達成，避免長談                  |

### 4) AI 協作（離線簡版）

```
SYSTEM: 你協助將條列素材轉成 GOAR 提要，每段 2–3 句，避免誇飾。
INPUT: 今日事件、感官紀錄、路線節點。
OUTPUT: Goal/Obstacle/Action/Result 四段，語氣自然、資訊具體。
```

- 使用時機：咖啡停留與傍晚收尾各一次；不在路上邊走邊用。

### 5) 安全與節奏

- 僅帶必要裝備，手機不長時間外露；需要拍攝時靠牆或背對安全區域。
- 時間到就收：午後雨勢轉大立刻轉入室內，優先保留體力與素材安全。

## 結果 (Results)

| 指標           | 目標   | 實際      | 備註                           |
| -------------- | ------ | --------- | ------------------------------ |
| 可用觀察點     | ≥ 10   | 13        | 去重後 11 可延展               |
| 三段式敘事草稿 | 1 版   | 1.1 版    | 晚上僅微調語氣                 |
| 步行距離       | 6–8 km | 約 7.1 km | 海拔影響步速，節奏仍穩         |
| 干擾事件       | ≤ 3    | 2         | 一次是雨勢，一次為人流密集改線 |

質性觀察：

- 舊城區的層次很適合「感官 → 敘事」的轉換練習，素材密度高但需留意節奏切換。
- IHG 的硬體與位置讓第一晚的身心回到穩定基線，之後更容易進入輸出模式。
- AI 的角色在今天是「提要器」而不是「代筆」，讓文章仍保有我的語氣與視角。

---

## 小結

抵達的第一晚與舊城區的一日，不追求爆量景點，而是把自己調回「能寫、能觀察、能延展」的狀態。把目標切小、把阻礙量化、把行動簡化、把結果留痕，之後的每一天就能靠著這個節奏穩定累積。
