#### 延伸閱讀

中文符號打法
```
※ 本文介紹之方法，適用環境為PC，中文輸入模式。

特殊常用按鍵：重音符 「`」
「`」通常位於鍵盤的左上方，Esc鍵下方，

常用的波浪符號「~」就是使用「Shift + ` 」來打印
```

本文中介紹的許多快速鍵皆為使用重音符的組合鍵

1. 逗號 ，
「 ` + , 」

2. 句號 。
「 ` + . 」

3. 驚嘆號 ！
「 Ctrl + Shift + 1」

4. 頓號 、
「 ` + ‘ 」

5. 分號 ；
「 ` + ;」

6. 冒號 ：
「 Ctrl + Shift + ;」

7. 上括號 「
「 Ctrl + Alt + ,」+「=」

8. 下括號 」
「 Ctrl + Alt + ,」+「\」

9. 括號 《
「 Ctrl + Alt + ,」+「u」

10. 括號 》
「 Ctrl + Alt + ,」+「i」

11. 括號 【
「 Ctrl + Alt + ,」+「o」

12. 括號 】
「 Ctrl + Alt + ,」+「p」

13. 全形等號 ＝
「 ` + =」

14. 全形斜線 ／
「 ` + /」
