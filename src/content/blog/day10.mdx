### 正文開始

然後，今天我們要來做個人品牌建立。

首先先了解自己的優勢：
```
我：
國立大學畢業，海外軟體工作室經驗五年。資訊學術背景，英語能力流利。
```

我們先不要點擊 [這裡](/blog/day8) 回去看，而是先相信自己的社會經驗，和大腦裡直覺告訴我們的資訊。

是不是覺得為什麼這位仁兄，還需要抽出時間來踏入自媒體的世界？

但是這個世界，就是比我們自己理解的都還要複雜。

曾經我們想要抽絲剝繭去 tagging 從各面向去嘗試了解這個社會。

但往往我們一開始看到的卻只是他人反映出來，表面上的光鮮亮麗。

於是落入了同溫層。<strike>(被演算法)</strike>

```
但是網路這個科技，讓我們有機會接觸異溫層，只要你願意繼續主動觸擊。
```

持續做你喜歡做的事情，十年磨一劍。（透過 0 與 1 的重構，每個人都可以有機會可以活成另一個形狀。）

而在國外這幾年旅居的經驗，也告訴我，永遠不要放棄讓自己跟外界接觸的機會。
