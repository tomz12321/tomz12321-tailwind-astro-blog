### 正文開始
了解生活成本 (住宿 水電 網路 食材)

![image](/static/images/Day5/myMeal.png)
這是我通常進入工作室的第一餐。

舉例附上我的生活成本 。

```
1100 通勤 (走路 $0)
1230 進入工作室 (第一餐，約 $150) // 早上的第一餐，健康飲食很重要！
1630 寫作時刻 (第二餐，約 $110) // $100~$120 外食
1830 晚餐時刻 (第三餐，約 $250) // 照顧自己的心理需求－胡蘿蔔加大棒（英語：Carrot and Stick）
2000 離開工作室 (日租，約 $805)
0030 水電瓦斯/網路計算 (每日，約$65)
```

我的工作一日花費：約 $1380。 (我的生存線，出現了！)

但是我的食<strike>衣</strike>住行只解決了三個，我接下來同樣的衣服可能要一直穿下去了。(崩潰 =口=!!!)
